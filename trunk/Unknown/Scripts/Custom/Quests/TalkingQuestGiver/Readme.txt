Description:
Creates a NPC that will generate quests when a player talks to him. This is a global quest that anyone can do at any time.

Installation:
Drop it in your Custom folder.
Save the TalkingNPXList.xml to your /DATA folder
Restart the server.
[Add QuestGiver

NOTE: If you are using any of my other NPC script you will need to upgrade them by download my TalkingNPCPack
